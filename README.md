# Servico Facil

Sistema local em Python para gestão de serviços personalizados (ex: afiação, manutenção, costura).

## Funcionalidades
- Cadastro de clientes
- Cadastro de itens de serviço
- Criação e controle de ordens de serviço
- Status visual das ordens
- Contas a pagar
- Dashboard com gráficos

## Tecnologias
- Python 3
- SQLite
- Tkinter
- Matplotlib
- Pandas
